# cglantschnig.dev

This is my **private portfolio website**, built with **TanStack Start**, showcasing selected projects, experience, and technical focus areas. The site is designed to be fast, minimal, and developer-centric, with a strong emphasis on performance, maintainability, and modern web standards.

## Tech Stack

- **TanStack Start** – Full-stack React framework
- **React** – Component-based UI
- **TypeScript** – Type-safe development
- **TanStack Router** – Type-safe routing
- **TanStack Query** – Data fetching and caching
- **Modern CSS / Utility-first styling** – Responsive, mobile-first layout

## Features

- Server-side rendering for fast initial load and SEO
- Fully responsive design optimized for desktop and mobile
- Clean, minimal UI focused on content and readability
- Type-safe routing and data handling
- Easy to extend with new pages, projects, or experiments

## Development

```bash
npm install
npm run dev
```
